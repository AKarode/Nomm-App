# Nomm

## Setup Instructions

1. Create a virtual environment:
   ```bash
   python3 -m venv venv
   ```

2. Activate the virtual environment:
   ```bash
   source venv/bin/activate
   ```

3. Navigate to the backend directory:
   ```bash
   cd backend
   ```

4. Install the required dependencies:
   ```bash
   pip3 install -r requirements.txt
   ```
